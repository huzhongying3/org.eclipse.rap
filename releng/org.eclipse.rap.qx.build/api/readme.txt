This snapshots directory contains API snapshots of milestone releases made 
with the PDE UI Incubator ApiTools.
Further information on ApiTools can be found here:
  http://wiki.eclipse.org/index.php/PDE_UI_Incubator_ApiTools_Compare
  
Currently the following RAP projects are included in the API snapshot
- org.eclipse.rap.help
- org.eclipse.rap.jface
- org.eclipse.rap.rwt
- org.eclipse.rap.ui.workbench
- org.eclipse.rap.w4t
